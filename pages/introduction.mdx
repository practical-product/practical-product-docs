# 프로젝트 소개

## 개요

Practical Product는 패션 카테고리를 중심으로 한 커머스 서비스입니다.
디자인은 무신사를 참고하되, 백엔드는 실무에서 자주 마주치는
"결제·재고·인증" 문제를 직접 풀어보는 데 초점을 맞췄습니다.

## 주요 기능

### 쇼핑 경험
- 카테고리 기반 상품 탐색, 검색
- 상품 옵션(SKU) 기반 재고 관리 — 사이즈/컬러 등 옵션 조합별 재고 추적
- 장바구니 (Zustand 기반 클라이언트 상태 관리)

### 주문 · 결제
- 재고 예약 기반 주문 생성 (원자적 차감으로 동시 주문 시 초과 판매 방지)
- **토스페이먼츠(Toss Payments)** 연동 결제 승인
- 가상계좌 결제 및 입금 완료 웹훅 처리
- 결제 취소/환불과 주문 상태 머신 관리
- 커서 기반 페이징으로 내 주문 목록 조회

### 회원
- httpOnly 쿠키 기반 인증 + 리프레시 토큰 로테이션
- 리프레시 토큰 재사용 탐지 (탈취 시 즉시 무효화)
- 회원 정보 조회/수정, 소프트 삭제 기반 탈퇴
- 개인정보(전화번호 등) 암호화 저장

### 운영
- Discord 알림 연동 (결제 실패율 등 비즈니스 지표 알림)
- GitHub Actions 기반 CI/CD, Docker로 EC2 자동 배포
- 코드 리뷰 자동화 (PR마다 빌드·테스트 실행)
