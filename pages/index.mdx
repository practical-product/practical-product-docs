# Practical Product

무신사(MUSINSA)의 UX를 참고해 만든 **패션 커머스 플랫폼**입니다.
상품 탐색부터 장바구니, 주문, 토스페이먼츠 결제까지 실제 커머스 서비스의
핵심 흐름을 처음부터 직접 설계하고 구현하는 것을 목표로 합니다.

## 왜 만들었나

단순한 CRUD 튜토리얼을 넘어, 실서비스에서 마주치는 문제들
(동시성 제어, 결제 정합성, 인증 보안, 관측성)을 직접 겪고 해결하기 위해
시작한 프로젝트입니다.

## 핵심 포인트

- **모듈러 모놀리식 백엔드** — 도메인별로 모듈을 분리해 응집도와 독립성을 확보
- **결제 신뢰성** — 토스페이먼츠 연동, 웹훅 + Outbox 패턴으로 이벤트 유실 방지
- **재고 동시성 제어** — 원자적 재고 예약/해제로 초과 판매 방지
- **가시성 확보** — Prometheus·Grafana·Loki·Tempo로 운영 상태를 실시간 추적

자세한 내용은 [프로젝트 소개](/introduction)와 [아키텍처](/architecture)에서
확인하실 수 있습니다.
