# 기술 스택

## 백엔드

- **언어/프레임워크**: Java, Spring Boot
- **빌드**: Gradle (멀티모듈)
- **데이터베이스**: MySQL
- **결제**: 토스페이먼츠(Toss Payments)
- **인증**: JWT, httpOnly Cookie
- **관측성**: Prometheus, Grafana, Loki, Tempo
- **알림**: Discord
- **배포**: Docker, GitHub Actions, EC2

## 프론트엔드

- **프레임워크**: Next.js (App Router)
- **상태 관리**: Zustand
- **테스트**: Playwright (E2E), Vitest
- **패키지 관리**: pnpm workspace

## 협업 · 자동화

- GitHub Actions CI/CD
- PR마다 자동 빌드·테스트 실행하는 코드 리뷰 봇
- 이 문서 사이트: **Nextra + Vercel**
